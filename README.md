# AutoSaline - 沿海社区清洁水源解决方案

一个为偏远沿海社区设计的太阳能海水淡化设备展示网站。

## 项目概述

AutoSaline 是一个紧凑型、太阳能驱动的海水淡化设备，专为面临淡水短缺的偏远沿海家庭、小型企业和政府而设计。本网站展示了该创新产品的技术特点、团队信息、市场影响力和商业模式。

## 使用指南

### 本地运行
1. 克隆或下载项目文件
2. 使用现代浏览器打开 `index.html`
3. 无需服务器，直接运行

### 语言切换
- 点击导航栏右上角的语言切换按钮（EN/中文）
- 语言偏好会自动保存到本地存储
- 刷新页面后保持上次选择的语言

### 浏览网站
- 使用顶部导航栏访问不同页面
- 点击页面中的按钮进行页面跳转
- 在首页滚动查看完整内容和动画效果
